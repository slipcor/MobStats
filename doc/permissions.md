# Permission Nodes

The following nodes can be used:

Node |  Definition | Default
------------- | ------------- | -------------
mobstats.config | read and alter config values | 游리 op
mobstats.count | have your kills counted | 游릭 true
mobstats.debug | start and stop debugging | 游리 op
mobstats.migrate | migrate from one database type to another | 游리 op
mobstats.opmessages | receive OP warnings when something is going wrong | 游리 op
mobstats.purge | purge older entries | 游리 op
mobstats.reload | reload the plugin | 游리 op
mobstats.top | get other player's statistics | 游릭 true
mobstats.nonewbie | prevent others from scoring kills by killing you | 游릭 true
mobstats.wipe | wipe statistics completely | 游리 op
